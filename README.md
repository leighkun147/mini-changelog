## The "Zero-Loop" Engineering Challenge
This version (v0) was built under strict architectural constraints to demonstrate fundamental logic mastery:
- **Zero Loops**: No `for` or `while` loops were used.
- **Zero Lists**: No `[]` or list-based data structures were used.
- **TDD Approach**: 11 automated test cases ensure 100% compliance with the technical specification.
